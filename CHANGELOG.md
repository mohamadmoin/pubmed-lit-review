# Changelog

## [0.1.0] - 2026-07-22

### Added

- PubMed AI literature review pipeline (search, full text, LLM synthesis)
- Django REST API with token authentication and Celery workers
- Neo4j graph storage with hybrid filesystem for large texts
- Flutter client for document generation, progress tracking, and reading
- Docker Compose deployment stack
- CLI for headless API access
- Bundled Flutter web app served at `/` (no Flutter install for end users)
- One-click launch scripts (`scripts/start.sh`, `scripts/start.bat`)
- Versioning system (`VERSION`, `scripts/release.sh`, `docs/RELEASE.md`)
- User setup guide (`docs/GETTING_STARTED.md`)
